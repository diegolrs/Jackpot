# Jackpot
## The classic jackpot machine made with Java

### This project uses the libraries:

* Color;
* Graphics;
* Random;
* BufferedImage;
* BufferStrategy;
* ImageIO;

### and the main class was extended by Canvas and implemented by Runnable and MouseListener, anothers Java's natives features.
